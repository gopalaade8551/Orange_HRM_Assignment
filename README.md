This project is related to the Orange HRM Login and Adding the employee into the list.
Here I have created Hybrid Framework with Selenium and TestNg with the Help of Page Object Model Design Patter.
The Framework structure includes various folders include PageObjects, Test Base, Test Cases, Test data,utilities.
The page object folder consist of classes for each page with the respective webelemets and their action ,methods.
The test case folder consist of test cases which covers the end to end test case of login the user, adding the employee into the list and checking the employee.
Test Case Description:
1. User should logi with the valid credentials.
2. 2.Click on PIM and cllck on add employee option.
3. employee should be added with the respective firstname, lastname and other details and click on save.
4. click on employee list and list should display all the saved employees.
